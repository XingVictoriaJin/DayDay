##Spring Boot: JPA, MyBatis, Redit, WebSocket
##Jedis: distributed session

BackEnd: SpringBoot, Seller side; 
Restfull API
Nginx, Tomcat, Redis, MySQL

Buyer(phone):

2.query order
3.cancel order

Seller(PC):

4. search order
5. cancel order
